• :active - until mouse is clicked
  • :checked - until checkbox or radio is checked
  • :default - the input that comes checked by default with the checked attribute
  • :dir(ltr | rtl) - get the text writter right to left or ltr
  • :disabled - get all the disabled items with the disable attribute
  • :empty - get the elements that dont have child(includes enter as a child)
  • :first-child - get the first child of the element
  • :first-of-type - get the first element hat enter in the descriptions
  • :fullscreen - when the element is in fullscreen
  • :focus - select the input that gets focused
  • :hover - by passing mouse over the element(on click at cellphones)
  • :in-range - when element value is within the range limits of min and max(number and date) or :out-of-range for inverse
  • :invalid - when the element don't pass on the input type validation or :valid for the ones that pass
  • :link - get all links within the element
  • :not() - get all elements that are not in that specification
  • :only-child - get all elements that don't have siblings
  • :optional - get input, select and textarea elements that don't have the required attribute
  • :read-only - get all the read-only elements, used for form-items or :read-write for the writable elements
  • :required - for the elements with the required attribute
  • :root - same as html element, used to declare variables in css
  • :target - when a link scroll the page to an element, get that element
  • :visited - get all the visited links
