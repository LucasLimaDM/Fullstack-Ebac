• ::after - create a pseudo-element that is the last child of the selected element, use ::before for first child
  • ::backdrop - background of fullscreen and modal elements
  • ::cue - get the caption of a video player, use ::cue-region for some other things
  • ::file-selector-button - get the button of a input:file
